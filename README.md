# Personal Portfolio — Landing Page

A clean, responsive personal portfolio landing page built with plain HTML, CSS, and JavaScript. No frameworks, no build tools — just the web platform.

---
*Project live at  https://jay-68.github.io/personal-landing-page/

## 📁 Project Structure

```
portfolio/
├── index.html          # Main entry point
├── css/
│   └── styles.css      # All styles
├── js/
│   └── main.js         # Interactivity & animations
└── assets/
    ├── images/         # Photos, project screenshots
    └── icons/          # Favicons, SVG icons
```

## 🚀 Setup & Installation

No dependencies or build step required.

**1. Clone the repository**

```bash
git clone https://github.com/your-username/your-portfolio.git
cd your-portfolio
```

**2. Open locally**

Simply open `index.html` in your browser:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```
## 🤝 Contributing

Contributions, fixes, and suggestions are welcome!

**1. Fork the repository** and create a new branch:

```bash

```

**2. Make your changes**, keeping the following in mind:

- Maintain consistent indentation (2 spaces)
